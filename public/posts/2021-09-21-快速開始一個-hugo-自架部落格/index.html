[2021-09-21] 快速開始一個 Hugo 自架部落格

2020-09-21 20:28:23 UTC+8 分類 hugo

紀錄自己的筆記讓自己回頭來看更容易統整

什麼是 Hugo

It’s open-source, 開放原始碼

A Framework, 一種框架工具

優點：

  • 快速的網站打包速度
  • 快速產生靜態網頁
  • 從無到有做一個網站的速度 ( 前提：你需要先學會他的一些操作語法 )
  • 官方上提供的各種佈景 ( 前提：你必須學會該怎麼使用這些 )
  • 做到「擁有一個寫文環境」

缺點：

  • 不像是其他部落格有個方便的後台管理
  • 需要自己找到可以部署的地方

安裝 Hugo

前置作業

  • 安裝 git
  • 安裝 golang
  • 安裝 hugo
